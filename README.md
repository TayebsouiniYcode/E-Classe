### Technologies utilisés : 
    
* HTML 5 
* Bootstrap v5.0.2
* CSS just pour personalisé des parties du code
* php 8
* JSON files (students data)
* CSV files (payments data)
* Associative array

* Réalisation de l'UI adaptable de la maquette avec un framework CSS (Bootstrap)
